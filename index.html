Linked Connections

Scalable intermodal route planning

A hypermedia framework for public transit route planning

  • Basics

Step 1: Click a pin on the map


Linked Connections requires pages of connections ordered by departure time. This way, a route planning algorithm can go through a window of time once and build a minimum spanning tree. By using Linked Data, machines can automatically travers the Web for route planning advice.

Tools

lc-server

lc-client

CSA.js

The connection scan algorithm (csa) for javascript. Not a Linked Connections specific tool

gtfs2lc

Transform your GTFS files in Linked Connections and load it into a store
