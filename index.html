Linked Connections

Scalable intermodal route planning

Linked Connections is a framework to publish transport data to the Web for route planning user agents

Publication

“Linked Connections define a way to publish raw transit data, so that it can be used for intermodal route planning.”
- ISWC2015 demo paper

Pieter Colpaert, Alejandro Llaves, Ruben Verborgh, Oscar Corcho, Erik Mannens, and Rik Van de Walle. Intermodal public transit routing using Linked Connections. Proceedings of the 14th International Semantic Web Conference: Posters and Demos (2015)

Tools

LinkedConnections/Server.js

a server which allows you to publish Linked Connections

LinkedConnections/Client.js

a client which allows you to plan routes over the Web from your own javascript application

CSA.js

The connection scan algorithm (csa) for javascript. Not a Linked Connections specific tool

gtfs2lc

Transform your GTFS files in Linked Connections and load it into a store
